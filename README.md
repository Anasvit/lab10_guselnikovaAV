# Лабораторная работа 10  Проверка данных при регистрации

## Описание проекта
Программа проверяет корректность данных, вводимых пользователем при регистрации:
логин (может быть телефоном, email или строкой), пароль и подтверждение пароля.

Реализовано более 10 различных проверок с выводом понятных сообщений об ошибках.


## Функциональность
- Валидация логина в формате телефона: `+x-xxx-xxx-xxxx`
- Валидация логина в формате email: `xxx@xxx.xxx`
- Валидация строкового логина: минимум 5 символов, только латиница, цифры и `_`
- Проверка логина по списку запрещённых (admin, root, guest и др.)
- Проверка пароля:
  - минимум 7 символов
  - только кириллица, цифры и спецсимволы (без латиницы)
  - обязательны заглавная и строчная буквы, цифра, спецсимвол
- Проверка совпадения пароля и подтверждения
- Маскирование паролей при логировании
- Логирование успешных и неуспешных попыток регистрации

## Используемые технологии
- Язык: C#
- Среда разработки: Visual Studio



## Примеры работы

| `admin`        | `Пароль1!`  | False — логин запрещён |
| `+7-999-123-45-67` | `Пароль1!` | False — неверный формат телефона |
| `user@mail.ru` | `Пароль1!`  | True |
| `abc`          | `Пароль1!`  | False — короткий логин |
| `my_login`     | `Pass1!`    | False — латиница в пароле |
| `my_login`     | `пароль`    | False — нет цифры и спецсимвола |
| `my_login`     | `Пароль1!`  | True |
| `my_login`     | `Пароль1!` / `Пароль2!` | False — пароли не совпадают |
